Напиши программу, для заполнения нашего журнала excel из регистраационной карточки landocs
Файл excel находится по ссылке:
\\Prim-fs-serv\primrdu\СРЗА\Журнал регистрации корреспонденции\Журнал регистрации входящей документации.xlsx
со структурой столбцов:
Дата	№ вх	№ письма	Тема письма	Автор письма	За подписью	№ папки	Кто регистрировал	Ключевые слова	Связанное письмо
новые данные вводятся в последнюю пустую строку на последнем листе файла


Программа запускается по hotkey или из ПКМ как-нибудь еще быстро. Когда она запускается уже активно окно регистрационной карточки в landocs и выдерено первое поле формы (назовем его 0 Tabов).

далее данные берутся Tabами из ячеек, 
Дата	№ вх	№ письма	Тема письма	Автор письма	За подписью	№ папки	Кто регистрировал	Ключевые слова	Связанное письмо
6	0	7	11	Вводится пользователем	9+5 (Подписант+Корреспондент)	Выбор пользователя	Взять имя пользователя windows	Вводится пользователем	16
после этого выводится окошко с тремя полями для ввода текста, кнопкой для ввода адреса (см подробнее ниже) и кнопкой "зарегестрировать"
Есть поля, в заполнении которых нужны действия пользователя:
Автор письма - просто ввод текста пользователем, по умолчанию стоит "-"
Ключевые слова -  просто ввод текста пользователем, по умолчанию пусто
Название письма - просто ввод текста пользователем, по умолчанию сторится как "[дата в формате гггг-мм-дд из поля Дата] [№ вх]_[№ письма(если в № есть запрещенные символы, например "/", то заменяем их на "_")]_[дата в формате дд_мм_гггг из поля Дата]"
Выбор папки скачивания - в поле №4 по Tabам находится ссылка на файл письма. при нажатии на кнопку Выбор папки скачивания открывается стандартная форма сохранения файла windows, начинающаяся с адреса "\\Prim-fs-serv\primrdu\СРЗА\Дела СРЗА\19 Переписка", название файла берется из "Название письма"
В поле "№ папки" вставляется часть пути после части по-умолчанию, то есть например путь "\\Prim-fs-serv\primrdu\СРЗА\Дела СРЗА\19 Переписка\19.7 По расчету уставок\2026", "№ папки"="19.7 По расчету уставок\2026"
Когда получены все данные нажимается кнопка "Зарегестрировать в журнал"

И данные регестрируются в журнал
\\Prim-fs-serv\primrdu\СРЗА\Журнал регистрации корреспонденции\Журнал регистрации входящей документации.xlsx

Дата	№ вх	№ письма	Тема письма	Автор письма	За подписью	№ папки	Кто регистрировал	Ключевые слова	Связанное письмо
6(в формате ггг-мм-дд)	0	7 (с гиперссылкой на письмо)	11	Из поля	9+5 (Подписант_[перенос на след строку в рамках одной ячейки]_Корреспондент)	Из поля	Взять имя пользователя windows	Из поля	16
новые данные вводятся в последнюю пустую строку на последнем листе файла. После выполнения файл excel не закрывается

Программа далжна запускаться на моем рабочем компьютере где нет доступа в интернет, поэтому зависимости должны быть минимизированны или запакованны, в общем в идеале вообще exeшник
